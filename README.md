# Literaku

**Literaku** is an innovative application designed to assist and enhance the literary skills of
people with disabilities, particularly those with blindness. Leveraging human interaction, the app
utilizes voice commands to facilitate functions, allowing literature such as books, journals, and
papers to be accessible through text-to-speech and controlled via speech-to-text technology.

## Features

- **Text-to-Speech Integration**: Converts written text into spoken words, enabling users to listen
  to literary content effortlessly.

- **Speech-to-Text Control**: Empowers users to navigate and interact with the application using
  voice commands, facilitating a seamless user experience.

- **Enhanced Accessibility**: Tailored specifically for individuals with disabilities, Literaku
  provides features aimed at improving accessibility and usability.

- **Local History Keeper**: Keeps track of previously accessed content, allowing users to easily
  revisit materials and track their literary journey.

- **Collection Tab**: Organizes literature into collections, facilitating efficient management and
  navigation of content.

- **Google PDF Database Search**: Allows users to search for books based on a database of PDFs from
  Google, expanding the range of available literary materials.

- **Updated User Interface**: Boasts a modern and intuitive interface, enhancing overall user
  experience and accessibility.

- **Video Tutorials**: Provides comprehensive video tutorials to guide users through the app's
  features and functionalities.

- **Expanded Command Set**: Includes an expanded set of voice commands to significantly enhance the
  quality of life for people with blindness, enabling them to navigate and interact with literary
  materials more effectively.

## Technology Stack

- **Framework**: Flutter
- **Programming Language**: Dart

## Legacy and Enhancements

Literaku represents an evolution from its predecessor, Literaku Legacy, which was initially built
using Kotlin and Android Native. The transition to Flutter and Dart brought about significant
enhancements, including the introduction of local history tracking, a collection tab for content
organization, an updated user interface, comprehensive video tutorials, and an expanded set of voice
commands. These improvements aim to provide a more robust and accessible platform for individuals
with disabilities to engage with literature effectively.
